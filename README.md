# linux-setup

My linux setup

## Quick Install

```bash
command -v sudo >/dev/null 2>&1 || su -c "apt-get update && apt-get install -y sudo" && sudo apt-get update && sudo apt-get install -y git && git clone https://github.com/bibliothek/linux-setup.git && cd linux-setup && ./run.sh
```
